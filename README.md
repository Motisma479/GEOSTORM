# GeoStorm

This Project was an introduction to c# and used raylib for the rendering. we have a week to work on a game like Geometry Wars.
## Controls :

- Default :
    | First Header  |     Action    |
    |     :---:     |     :---:     |
    |       W       |       Up      |
    |       A       |      Left     |
    |       S       |      Down     |
    |       D       |     Right     |
    |     Mouse     |  Move Cursor  |
    |   Left Click  |     Shoot     |
    |      Esc      |     Pause     |

Controls can be modified in settings.
